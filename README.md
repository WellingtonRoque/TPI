# 👨‍💻 Sobre a Disciplina

Bootcamp focado no desenvolvimento de sistemas utilizando:

- Programação Orientada a Objetos
- Arquitetura de Software
- MySQL
- MVC
- CRUD
- Versionamento com Git

A disciplina será totalmente prática e baseada no desenvolvimento de um projeto real.
